# PouliDAO-SC

## Usage
### Installation
```shell
npm install
npx hardhat compile
npx hardhat test
```
## Developer Protal
https://developer.worldcoin.org/login

### Deployments
Mock deployment:
`npx hardhat run scripts/MockDeployments.ts [--network <your_network>]`

Integrated deployment:
`npx hardhat run scripts/deployments.js --network <your-network>`
