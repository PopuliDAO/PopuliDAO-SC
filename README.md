# PouliDAO-SC

## Installation
```shell
npm install
npx hardhat compile
npx hardhat test
```

## Deployment

Integrated deployment
`npx hardhat run scripts/deploy.js --network <your-network>`

Mock deployment
`npx hardhat run scripts/MockDeployments.ts [--network <your_network>]`
